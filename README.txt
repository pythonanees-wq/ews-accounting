FREE HOSTING - STEP BY STEP GUIDE
=====================================================

OPTION 1: GITHUB PAGES (Recommended)
-------------------------------------

Step 1: Create GitHub Account
  1. Go to https://github.com
  2. Click "Sign up" and create free account
  3. Verify your email

Step 2: Create Repository
  1. Click "+" button (top right) -> "New repository"
  2. Name it: ews-accounting
  3. Select "Public"
  4. Check "Add a README file"
  5. Click "Create repository"

Step 3: Upload Files
  1. In your repository, click "Add file" -> "Upload files"
  2. Upload ALL files from this folder:
     - index.html
     - manifest.json
     - sw.js
     - icons/ folder (with all PNG files)
  3. Click "Commit changes"

Step 4: Enable GitHub Pages
  1. Go to repository "Settings"
  2. Click "Pages" in left sidebar
  3. Under "Source", select "main" branch
  4. Click "Save"
  5. Wait 2-3 minutes

Step 5: Access Your App
  Your app will be at:
  https://YOUR-USERNAME.github.io/ews-accounting/

-------------------------------------
OPTION 2: NETLIFY (Easiest)
-------------------------------------

Step 1: Go to https://app.netlify.com/drop
Step 2: Drag the entire pwa_accounting folder onto the page
Step 3: Done! You get a URL like: https://random-name.netlify.app

=====================================================
INSTALLING ON YOUR PHONE
=====================================================

ANDROID:
  1. Open Chrome browser
  2. Go to your app URL
  3. Tap "Add to Home screen" banner (or menu -> Add to Home screen)
  4. App icon appears on home screen!

IPHONE/IPAD:
  1. Open Safari browser
  2. Go to your app URL
  3. Tap Share button
  4. Tap "Add to Home Screen"
  5. Tap "Add"

=====================================================
DATA BACKUP
=====================================================

1. Go to Backup/Restore section in app
2. Click "Download Backup"
3. Save JSON file to Google Drive

To restore: Select file and click "Restore Backup"
